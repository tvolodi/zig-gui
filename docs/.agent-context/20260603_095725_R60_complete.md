## Key design decision

`layoutParagraph` signature was NOT changed (no 7th `variant` parameter). Instead, `variant` is stored on `Font.variant` and read from there. This avoids breaking `test-02` which calls `layoutParagraph` with 6 arguments. The R60 spec noted this was an "open question" requiring human confirmation — storing variant on `Font` achieves the same semantic separation (correct atlas keys) without a breaking signature change.
