## What was built

**R90 — Debug Overlay (`src/app/debug_overlay.zig`)**  
New `DebugOverlay` struct toggled by F1. When enabled, draws colored `border_rect` commands over every live scene element (accent = hovered, info = focusable, ok = container, warn = other) and a four-line info panel for the hovered element. `updateHover` iterates elements in reverse painter order; `buildDebugDrawList` returns a heap-allocated `[]DrawCommand` slice.

**R91 — Scene Dump (`src/07/debug.zig`)**  
New file with two free functions: `debugPrintScene` (DFS pre-order, 2 spaces/level, element rect + style summary + flags) and `debugPrintSceneStats` (one-line: live/total/dirty/focused). Both write to stderr using a 256-byte stack buffer — no heap allocations. Forwarded via `Scene.debugPrint()` and `Scene.debugPrintStats()`.

**R92 — Performance HUD (`src/app/perf_hud.zig`)**  
New `FrameCounters` struct (frame_ms, cmd_count, dirty_count, element_count) and `PerfHud` struct (16-entry ring buffer, smoothed FPS, 3-line top-right panel). Active only when debug overlay is enabled. AppInner records counters after each `endFrame`.

**R93 — Theme Live-Swap (`src/app/app.zig`)**  
Added `AppInner.setTheme(theme)`, `toggleTheme()`, `rebuildStyles()`, and `resolveStyleForIdx`. `setTheme` scales tokens by the current font factor, re-resolves all live element styles from their stored CSS class strings, and marks all elements dirty. The `_current_palette` and `_current_mode` fields persist the current theme context.

**R94 — Font Scaling (`src/app/app.zig`, `src/05/types.zig`)**  
Added `Tokens.scaled(factor)` (pure, clamps text sizes to [6,96]). Added `AppInner.setFontScale(factor)` and `getFontScale()`. Factor is clamped to [0.5, 4.0], stored as `_font_scale`, and applied to the token set on every theme rebuild.

**R95 — High-Contrast Mode (`src/05/types.zig`)**  
Added `Palette.highContrast()` (WCAG AA light: accent_400 = 0x0055CC, body text = black) and `Palette.highContrastDark()` (WCAG AA dark: accent_400 = 0xFFCC00, body text = white). Added `Theme.hc_light` and `Theme.hc_dark` convenience constants.
