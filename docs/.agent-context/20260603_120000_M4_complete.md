## What was built

### R40 — Pseudo-state styling
- `PseudoState` packed struct (hover/focus/active/disabled flags) added to module 07.
- `_pseudo[]` parallel array in `Scene`; `setPseudo(idx, state)` marks element dirty.
- `PseudoStyleSet` / `PseudoOverride` types in module 05; `buttonPseudo`, `inputPseudo`,
  `dropdownPseudo`, `checkboxPseudo` builders return per-kind override sets.
- `resolveStyle(base, overrides, state)` in module 09 layers overrides in priority order:
  focus < hover < active < disabled. Called per-element during `buildDrawList`.

### R41 — Overlay / z-layer
- `OverlayLayer` in `src/app/overlay.zig`: ordered list of named `DrawCommand` slices.
- API: `allocId` → `setSlot` → `flatten` → submit after main draw list; `removeSlot` on dismiss.
- `OverlayId = u16`; slots rendered in insertion order.

### R42 — Clipping / overflow-hidden
- `rectToScissor` and `intersectScissor` helpers in module 09.
- `buildDrawList` emits a `scissor` command when entering a `scrollview`; children rects are
  offset by scroll position; `restore_scissor` sentinel pushed after children.
- `defaultLayoutFor(.scrollview)` sets `overflow = .hidden` automatically.

### R43 — Image / icon rendering
- `ImageAtlas` (CPU-side RGBA shelf-packing atlas, 512×512) in `src/app/image_atlas.zig`.
- `addImage(pixels, w, h) → ImageId`; `getRect(id) → ImageRect` (normalized UVs).
- `ImageState` parallel array `_image_state[]` in `Scene`; `setImage` / `setImageTint` APIs.
- `GpuImageAtlas` stub in module 09 (real Vulkan upload deferred to GPU integration step).
- `<Image>` and `<Icon>` widget kinds registered in `tagToKind`; tint applied for icons.

### R44 — Text truncation with ellipsis
- `truncate: bool` field added to `ComputedStyle`; resolved by `resolveClasses` from
  Tailwind class `truncate`.
- `buildDrawList` checks `style.truncate`: clips glyph commands and appends ellipsis
  when text would overflow the element's computed width.

### R45 — Opacity
- `opacity: f32` field in `ComputedStyle` (default 1.0); resolved from `opacity-{0,25,50,75,100}`.
- `applyOpacity(col, factor)` helper in module 09.
- `buildDrawList` accumulates `effective_alpha = parent_alpha * style.opacity` through the
  DFS stack and applies it to every color emitted for that element and its descendants.

### R46 — Box shadow
- `shadow_blur`, `shadow_offset_x`, `shadow_offset_y`, `shadow_color` fields in `ComputedStyle`.
- Resolved from Tailwind `shadow-sm` / `shadow` / `shadow-md` / `shadow-lg` / `shadow-xl` /
  `shadow-none` classes.
- `emitShadow(...)` emits 5 concentric `filled_rect` commands before the element background.
  Skipped when `shadow_blur == 0`.
