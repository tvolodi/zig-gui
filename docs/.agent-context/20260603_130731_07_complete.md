# Module 07 Phase 3 — Completion Summary

## What was built

Phase 3 (Complex Widgets) of Milestone 7 added five new widget kinds to the `src/07/types.zig` Scene and corresponding render cases in `src/09/types.zig`, plus three helper modules in `src/app/` and integration into `src/app/app.zig`.

### New widget kinds (added to `WidgetKind` enum)

- **`date_picker`** — ISO date display (R78). `DatePickerState` with `value: DateValue`, `nav_year/nav_month`, `open`, `disabled`. Parsed from `value="YYYY-MM-DD"` and `disabled=` attributes.
- **`avatar`** — circular user avatar (R7B). `AvatarState` with `image_id`, `initials: [2]u8`, `size_px: f32`. Rendered as image or colored circle with initials text.
- **`badge`** — notification pill (R7B). `BadgeState` with `count`, `visible`, `color`.
- **`data_table`** — virtualized data table (R79). `DataTableState` with column definitions (`DataColumn[MAX_COLUMNS]`), `sorted_indices: std.ArrayListUnmanaged(u32)`, `scroll_y`, `row_height`, and a `DataTableRows` callback pointer.

### New parallel arrays in `Scene`

- `_date_picker_state: std.ArrayListUnmanaged(DatePickerState)`
- `_avatar_state: std.ArrayListUnmanaged(AvatarState)`
- `_badge_state: std.ArrayListUnmanaged(BadgeState)`
- `_tooltip: std.ArrayListUnmanaged(?[]const u8)` — `null` = no tooltip; any element can have a tooltip
- `_context_menu_idx: std.ArrayListUnmanaged(u8)` — `0xFF` = no menu
- `_table_state: std.ArrayListUnmanaged(DataTableState)` — `sorted_indices` deinited per-element in `deinit`

### New accessor methods (Scene)

`datePickerStateOf`, `setDateValue`, `getDateValue`, `openCalendar`, `closeCalendar`, `avatarStateOf`, `setAvatarImage`, `setAvatarInitials`, `badgeStateOf`, `tooltipOf`, `setTooltip`, `contextMenuIdxOf`, `setContextMenuIdx`, `tableStateOf`, `setTableData`, `setTableColumns`, `sortTable`

### New helper modules

- `src/app/date_util.zig` — `daysInMonth`, `firstWeekday`, `formatDate`, `parseDate`
- `src/app/tooltip.zig` — `TooltipManager` with 500 ms hover delay, overlay rendering
- `src/app/context_menu.zig` — `ContextMenuManager` with registry, `openAt`, `dismiss`

### Renderer additions (`src/09/types.zig`)

- `initialsColor` helper — deterministic 8-color palette from char modulo
- `buildDrawList` cases for `.date_picker`, `.avatar`, `.badge`, `.data_table` (virtualized)

### App integration (`src/app/app.zig`)

- `AppInner` gains `tooltip_manager: TooltipManager` and `context_menu_manager: ContextMenuManager`
- `deinit` calls `tooltip_manager.deinit` and `context_menu_manager.deinit`
- `hasAnimatedElements` renamed to accept `*const TooltipManager` — returns true when `tooltip.isPending()`
- `dispatchEvents` mouse_move handler calls `onHover`/`onLeave` after hit-testing `_tooltip[]`
- `dispatchEvents` mouse_button right-click opens context menu via `openAt`
