### 9. App layer wiring (src/app/app.zig)
- Added `left_mouse_down`, `last_cursor_x`, `last_cursor_y` fields to AppInner
- Fully implemented `dispatchEvents` with:
  - Mouse hover → `updateHoverStates`
  - Mouse press/release → `handleMousePress` / `handleMouseRelease` (hit-test focusable elements)
  - Scroll → `handleScroll` (finds scrollview under cursor)
  - Key events → `handleKey` (Tab/Shift+Tab, Escape, per-kind dispatch)
  - Char events → `handleChar` (text insertion)
- Added `scene.fireQueuedCallbacks()` call after layout solve, before buildDrawList (INV-3.3)

### 10. Renderer additions (src/09/types.zig)

- Input cursor: 1.5px vertical FilledRect at computed cursor X position
- Input selection: semi-transparent FilledRect over selected range
- Checkbox: box outline (BorderRect) + filled inner square when checked
- Scrollbar: track + proportional thumb (right edge, 6px wide)
- Dropdown overlay: second pass draws option panel + labels above all other elements
- Added `computeTextX` helper for accurate cursor/selection positioning using atlas glyph widths
